# AiSchedule-for-hfut
小爱课程表 合肥工业大学适配

[官方开发文档](https://ldtu0m3md0.feishu.cn/docs/doccnhZPl8KnswEthRXUz8ivnhb)

## 文件说明

- **scheduleHtmlProvider.js**: 获取整个网页内容，默认传入document对象 

- **scheduleHtmlParser.js**: 解析html获取课表信息，组装并返回指定格式的 json 数据
